
Приложение для записи задач на неделю. 

Текущий день подсвечивается, остальные дни недели всегда отображаются как будущие дни. 
Когда создаётся задача, то в ней записывается и дата. Есть возможность добавлять задачи на 
срок дольше, чем 1 неделя - для этого на главной странице есть пропка "Upcoming tasks" - в
ней можно посмотреть все будущие даты, чей срок ещё не приблизился, а также добавить
новую дату на будущее.При переносе задачи на другой день, дата в ней также меняется.
При открытии приложения все задачи проверяются, и если есть задачи, дата которых уже в
прошлом, то все они помещаются в "outstanding Tasks", т.е. невыполненные задачи. Если дата
какой-то из "будущих задач" совпадает с датами дней недели на главной странице - то эта,
дата добавляется в соответствующий день недели. 

Все задачи можно перемещать в другие дни просто перетаскивая их. Кликнув по каждой задаче,
можно посмотреть её описание, а также редактировать задачу или удалить её. 

Кнопка "outstanding Tasks" горит серым цветом и некликабельная до тех пор, пока в ней нет
ни одной "просроченной" задачи.

Добавлена анимация плюсика "добавить задачу" дней недели при наведении на него, а также
анимация при наведение на плюсик "добавить задачу" будущих задач. В странице каждой будущей
задачи возле даты есть маленький календарик, на котором также добавлена анимация при
наведении. Кликнув по нему, можно изменить дату будущей задачи. В странице каждой
"просроченной" задачи есть кнопка "восстановить задачу", на которую также добавлена
анимация при наведении.

Добавить задачу можно при помощи клавиатуры - нажав на цифру от 1 до 7. Каждая цифра
соответствует порядковому номеру дня недели. Из каждой страницы можно вернуться назад при
помощи клавиши Backspace.
